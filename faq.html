Background

What is Kenya?

Kenya is an application runner that uses Maven metadata to provision artifacts and to assemble a runtime classpath. This allows a user to run a Java application without packaging it up if there is accurate Maven artifact metadata provided or discoverable.

[top]
Why the name Kenya?

The country of Kenya is known for its great runners. You saw that coming, right?

[top]
Should I use Kenya to run my production system?

Absolutely. Let us know how that goes... Seriously, Kenya should not replace building and packaging real Java applications, but it can be handy for simplifying the writing/deploying/running a certain class of application. Use the right tool for the job.
